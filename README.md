# 🚴 Subike

Sitio web estático desarrollado con HTML, CSS y JavaScript como proyecto práctico de formación.

## 📌 Descripción

Subike es una página web responsive enfocada en mostrar contenido visual atractivo, con diseño moderno, galería de imágenes y estructura adaptable a dispositivos móviles.

Este proyecto fue desarrollado para fortalecer habilidades en:

- Maquetación con HTML5
- Estilos avanzados con CSS3
- Diseño responsive
- Organización de estructura web
- Uso de Git y GitHub

---

## 🛠 Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Google Fonts
- Font Awesome
